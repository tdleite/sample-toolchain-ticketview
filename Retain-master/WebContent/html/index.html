Edit View Help

			

				
Queues

				
Search for all PMRs from the selected queue.


				
Engineers

				
Search for all PMRs from the selected engineer, who can be the owner, resolver or resolver5.


				
Status

				
Search for all PMRs from the monitoring queues that have the select status.

				
The status does not have any link to the PMR status on Retain. It's an internal status only used for management purposes.

				
Only status Done from the last week are selected.


				
PMR Attribute - Aged

				
Search for all PMRs from the monitoring queues that are aged.

				
Aged criteria varies according to the PMR severity (Sev 1 = 30 days, Sev 2 = 60 days, Sev 3 = 90 days, Sev 4 = 90 days).


				
PMR Attribute - Critsit

				
Search for all PMRs from the monitoring queues that Critical Situation flag.


				
PMR Attribute - Severity

				
Search for all PMRs from the monitoring queues that have the select severity.


				
Clients

				
Search for all PMRs from the monitoring queues that belongs to the selected client.

			

		

	


	

		

			
			
Edit Monitoring Queues Help

			

				
Available Queues

				
Shows all queues that are not selected.


				
Monitoring Queues

				
Shows all queues selected.

				
Columns Status, PMR Attribute and Clients shows only PMRs from the queues selected.
